# Favicon Files

This folder should contain the following favicon files for your Revu logo:

## Required Files:

1. **favicon-16x16.png** - 16x16 pixels
2. **favicon-32x32.png** - 32x32 pixels
3. **apple-touch-icon.png** - 180x180 pixels (for iOS devices)
4. **android-chrome-192x192.png** - 192x192 pixels
5. **android-chrome-512x512.png** - 512x512 pixels

## How to Generate These Files:

### Quick Method (Recommended):
1. Visit https://favicon.io/favicon-converter/
2. Upload your Revu logo (yellow background with microphone)
3. Download the generated package
4. Extract and place all PNG files here

### Manual Method:
Use an image editor to create each size from your logo image.

## Additional File:
- **favicon.ico** should be placed in the parent `assets/` folder (one level up)
